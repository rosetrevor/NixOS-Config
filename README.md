# My Dotfiles
These dotfiles are for my personal NixOS setup. These have been setup entirely while learning nix and linux in general, there are probably plenty of better way to do things. But they are available for any who may be interested.

## Configured Programs
My gloabl NixOS config is controlled via this repo across two machines. Much of this configuration is borrowed or based upon the contributions contributions others. These are a most of the things I have configured for my personal use:
- dunst
- hyprland ecosystem
- waybar
- kitty
- neovim
- tmux
- starship
- yazi
- ...and more!

I make no promises any of my configs will work for others, but feel free to reference them for your own needs.
